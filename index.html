Sistema Inteligente para Reconocimiento de Lenguaje de Señas

Una solución tecnológica para facilitar la comunicación de personas sordomudas mediante IA y modelos Transformer

Descripción del Proyecto

Propósito del Sistema

Este proyecto tiene como objetivo principal el desarrollo de un sistema inteligente que utilice tecnología de reconocimiento de lenguaje de señas en tiempo real, permitiendo la traducción instantánea de gestos de señas en texto o voz.

El sistema será capaz de reconocer signos de diferentes lenguajes de señas, adaptándose a las particularidades de cada contexto. Se utilizarán modelos de aprendizaje profundo basados en Transformers, conocidos por su capacidad para procesar secuencias de datos, lo cual los hace altamente eficientes para tareas como la interpretación del lenguaje de señas.

Problemática

El lenguaje de señas es una herramienta vital para las personas sordomudas, pero la falta de conocimiento y comprensión generalizada de este lenguaje por parte de la población en general crea una barrera significativa en la comunicación cotidiana.

En situaciones cotidianas como trámites administrativos, interacciones laborales o educativas, las personas sordomudas a menudo dependen de la ayuda de intérpretes, lo que no siempre es una opción viable. Este desafío se ve agravado por la escasez de tecnologías que proporcionen soluciones accesibles y efectivas en tiempo real.

Justificación

La inclusión social de las personas con discapacidad auditiva es una prioridad para garantizar su participación activa y equitativa en la sociedad. Sin embargo, las barreras de comunicación continúan siendo un obstáculo significativo.

La implementación de un sistema de reconocimiento de lenguaje de señas en tiempo real representa un paso importante para abordar esta desigualdad. Este proyecto no solo responde a una necesidad social urgente, sino que también aprovecha las capacidades de la inteligencia artificial, específicamente de los modelos Transformers, para proporcionar una solución técnica avanzada.

Objetivos del Proyecto

Objetivo Principal

Ofrecer una solución accesible y efectiva para que las personas sordomudas puedan comunicarse de manera fluida y eficiente con aquellas que no conocen el lenguaje de señas.

Objetivos Específicos

  • Desarrollar un modelo de reconocimiento de lenguaje de señas basado en Transformers para la traducción en tiempo real de gestos en texto o voz.
  • Crear una interfaz de usuario accesible y amigable para que tanto personas sordomudas como oyentes puedan interactuar fácilmente con el sistema.
  • Validar el sistema mediante pruebas en escenarios reales, para evaluar su precisión, facilidad de uso y efectividad en la mejora de la comunicación entre sordomudos y oyentes.
  • Optimizar el rendimiento del modelo de reconocimiento, para garantizar respuestas rápidas y precisas, incluso en condiciones variables de luz o contexto.

Metodología de Desarrollo

Enfoque Ágil

La metodología de desarrollo utilizada en este proyecto será Ágil. Se adoptará un enfoque iterativo e incremental, con un ciclo de desarrollo constante que se adapta a las necesidades emergentes del sistema.

Este enfoque permite realizar mejoras continuas y ajustar el sistema según los resultados obtenidos en las fases de pruebas y validación.

Etapas Principales

  • Planificación: Definición de requisitos y metas del sistema, asegurando que se comprendan correctamente los objetivos de inclusión y accesibilidad.
  • Desarrollo incremental: Desarrollo en ciclos cortos, con entrega de versiones funcionales del sistema en cada iteración.
  • Pruebas y validación continua: Cada versión del sistema será evaluada para asegurar que cumple con los objetivos establecidos y responde correctamente en tiempo real.
  • Retroalimentación y ajustes: Los resultados de las pruebas de los usuarios se utilizarán para realizar ajustes en el sistema, mejorando la precisión y la usabilidad del modelo.

Alcance del Proyecto

Componentes Principales

  • Modelo de IA: Implementación de un modelo de Transformer entrenado para reconocer signos del lenguaje de señas en tiempo real, utilizando secuencias de movimientos corporales capturados por MediaPipe.
  • Traducción al quechua: Creación de un sistema de traducción automático de español a quechua para que el sistema no solo reconozca las señas, sino que las traduzca al quechua en tiempo real.
  • Interfaz de usuario: Desarrollo de una interfaz accesible que permita la interacción entre personas sordomudas y oyentes, mostrando tanto las palabras reconocidas como sus traducciones.
  • Optimización en tiempo real: El sistema será capaz de procesar señales en tiempo real con una precisión mínima del 80% y tiempos de latencia bajos.
  • Generación de métricas: Sistema de métricas para evaluar el rendimiento del modelo en tiempo real, medir la precisión del reconocimiento y mejorar el sistema basado en los datos obtenidos.

Modelo de Datos

El modelo de datos estará compuesto por dos estructuras principales:

  • Diccionario de traducción: Estructura basada en JSON o bases de datos, que contiene la relación entre palabras en español y su traducción en quechua.
  • Secuencias de movimiento: Cada muestra de lenguaje de señas será representada como una secuencia de características de movimiento extraídas de los puntos de referencia de MediaPipe.
